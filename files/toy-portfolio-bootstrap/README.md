
# Toy Portfolio (Bootstrap)

This package contains a responsive Bootstrap 5 page for Arashdeep Singh's Toy Portfolio.

## How to run
1. Unzip `toy-portfolio-bootstrap.zip`.
2. Open `index.html` in any modern browser.
3. Replace placeholder images in `/images`:
   - `dummy1.jpg`, `dummy2.jpg`, `profile.jpg`, `headerback.png`

## Notes
- Bootstrap is loaded via CDN; internet connection is required for the navbar toggle and carousel JS/CSS.
- You can switch to local Bootstrap files if preferred.
